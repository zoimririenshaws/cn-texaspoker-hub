# cn-texaspoker-hub

## 项目简介

`cn-texaspoker-hub` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为集中存放和共享 HTML 资源的公共空间。

仓库中的页面内容可能与德州扑克相关，但并非所有页面均限定于此主题。每个 HTML 页面均为独立文件，可直接在浏览器中打开运行，无需额外依赖。

## 目录说明

```
cn-texaspoker-hub/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── LICENSE         # 开源许可证
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面。
- `assets/`：可选的资源文件夹，用于存放页面所需的公共资源。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，具备以下特点：

- **自包含**：每个页面不依赖外部服务器或数据库，所有逻辑与样式均嵌入或引用仓库内资源。
- **静态发布**：页面可直接通过 GitHub Pages 或其他静态托管服务部署。
- **无特定域名绑定**：页面内容不指向任何具体网站或域名，仅作为通用 HTML 资料存在。

如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保文件名简洁、无空格。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/cn-texaspoker-hub.git
   ```
2. 在浏览器中直接打开 `pages/` 下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 启用静态站点，将 `pages/` 作为根目录发布。

## 维护说明

- 本仓库接受合理的页面归档提交，但请确保提交的 HTML 内容不违反相关法律法规。
